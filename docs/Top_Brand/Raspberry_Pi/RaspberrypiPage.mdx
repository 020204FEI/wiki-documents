## Compare RPi Models
| Model    | Memory                                 | Form Factor | Ethernet | Wifi | Processor Type | Processor Speed |
|-------   |--------                                |-------------|----------|----- |----------------|-----------------|
| 3B+      | 512 MB RAM                             | Standard [a]          | ✅ |✅ |ARM1136JZF| 700MHz            |
| Zero     | 512 MB RAM                             | Ultra-compact [c]     | ❌ |❌ |BCM2835   | 1GHz              |
| Zero W/WH| 512 MB RAM                             | Ultra-compact [c]     | ❌ |✅ |BCM2835   | Dual Core @1GHz   |
| CM4      | 2/4/8 GB RAM up to 32GB eMMC storage   | Ultra-compact [c]     | ❌ |❌ |BCM2711   | Quad Core @1.5GHz |
