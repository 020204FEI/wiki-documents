## The reComputer series 

The reComputer series are hand-size edge AI boxes built with differenct NVIDIA Jetson module which delivers high AI performance to multiple edge scenarios. Each of it has a rich set of IOs, equipped with an aluminum case, a cooling fan with a heatsink, and a pre-installed JetPack System with NVIDIA JetPack™.
